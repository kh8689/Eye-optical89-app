# نظام إدارة عيادة فحص النظر

تطبيق ويب متكامل لإدارة عيادة العيون، مبني باستخدام React و Tailwind CSS.

## المميزات
- إدارة سجل المرضى.
- فحص النظر وإصدار الوصفات الطبية.
- إدارة المخزون (إطارات، عدسات).
- إصدار الفواتير وطباعتها.
- يعمل محليًا بالكامل (Local Storage).

## التشغيل المحلي

1. تثبيت المكتبات:
   ```bash
   npm install
   ```

2. تشغيل المشروع:
   ```bash
   npm run dev
   ```
